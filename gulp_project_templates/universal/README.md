#### Conception
- 這是一個可以搭載任何 HTML 專案的 Gulp Project Template

#### How to start

```
ln -sf $YOUR_HTML_APP_ROOT_PATH app
# ex: ln -sf ../../../kms-web-client/app app
```

```
npm install
npm install bower -gd
bower install
	... update gulpfile.js & recompile config rewrite part configuration.
gulp serve
```
